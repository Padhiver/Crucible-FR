# Glossaire complet du système de jeu Crucible

---

### Termes de Base

| Terme original | Traduction | Description |
| :--- | :--- | :--- |
| **Abilities** | **Scores de caractéristiques** | Les attributs physiques et mentaux d'un personnage, de 1 à 12. |
| **Actions** | **Actions** | Les actions qu'un personnage peut effectuer. |
| **Adversary** | **Adversaire** | Un ennemi ou un opposant. |
| **Hero** | **Héros** | Le personnage joueur. |
| **Item** | **Objet** | Un objet en jeu, comme une arme ou une armure. |
| **Sheet** | **Feuille** | La feuille d'un personnage ou d'un objet dans le système. |
| **Types** | **Types** | Les différents types d'acteurs (personnages, ennemis) ou d'objets. |

### Rangs de Menace (Adversary Threat Ranks)

| Terme original | Traduction | Description |
| :--- | :--- | :--- |
| **Minion** | **Créature mineure** | Ennemis plus faibles que la moyenne, mais souvent nombreux. |
| **Normal** | **Normal** | Le niveau de difficulté standard. |
| **Elite** | **Élite** | Ennemis plus forts et plus dangereux que la moyenne. |
| **Boss** | **Boss** | Ennemis particulièrement dangereux, le défi ultime d'un niveau. |

### Catégories d'Actions (Action Tags)

| Terme original | Traduction | Description |
| :--- | :--- | :--- |
| **Accurate** | **Précis** | Cette action est particulièrement précise. Vous gagnez 2 Bénédictions supplémentaires en l'exécutant. |
| **After Strike** | **Après la Frappe** | Cette action ne peut être effectuée qu'immédiatement après une attaque de frappe basique qui n'a pas été un échec critique. |
| **Dual Wield** | **Ambidextrie** | Cette action nécessite de manier deux armes distinctes ou d'être totalement à mains nues pour être exécutée. |
| **Ambush** | **Embuscade** | Cette arme est bien adaptée pour frapper avant que vos adversaires n'aient une chance de réagir. Lorsque vous attaquez une cible qui est plus bas que vous dans l'ordre de l'initiative, vous gagnez +1 Bénédiction. |
| **Brute** | **Brute** | Cette action implique l'application de la force brute et nécessite une arme qui évolue en utilisant la Force. |
| **Blocking** | **Blocage** | Cette arme fournit un bonus à votre défense de blocage. |
| **Consume** | **Consommer** | Cette action implique et nécessite la consommation d'un consommable spécifique. |
| **Deadly** | **Mortel** | Cette action a le potentiel d'infliger une quantité de dégâts énorme, augmentant le multiplicateur de dégâts de x1. |
| **Difficult** | **Difficile** | Cette action est compliquée ou difficile à exécuter. Vous subissez 2 malédictions supplémentaires en tentant l'action. |
| **Disarm** | **Désarmer** | Cette action, si elle réussit, fait tomber l'arme de la cible. |
| **Empowered** | **Renforcé** | Cette action est puissante et inflige des dégâts supplémentaires, gagnant un bonus de dégâts de +2 si elle réussit. |
| **Engaging** | **Engageant** | Les armes de cette catégorie vous permettent de mieux combattre plusieurs ennemis, augmentant votre engagement de +1. |
| **Finesse** | **Finesse** | Cette action nécessite une certaine finesse pour être exécutée. Vous devez manier une arme qui évolue en utilisant la Dextérité. |
| **Flanking** | **Prise en tenaille** | Cette action ne peut être effectuée que contre une cible prise en tenaille ou non avertie. |
| **Freehand** | **Main libre** | Cette action nécessite au moins une main qui ne tient pas d'objet pour être exécutée. |
| **Generic** | **Générique** | Actions qui effectuent une attaque générique sans utiliser de sort ou d'arme spécifique. |
| **Harmless** | **Inoffensif** | Cette action n'impose pas d'effet directement nocif, son multiplicateur de dégâts est nul. |
| **Healing** | **Guérison** | Cette action offre une restauration en testant le seuil de blessures d'un allié. |
| **Keen** | **Acéré** | Cette arme est particulièrement mortelle lorsqu'elle atteint un coup critique. |
| **Mechanical** | **Mécanique** | Cette action nécessite l'utilisation d'une arme mécanique. |
| **Melee** | **Mêlée** | Cette action nécessite de manier une arme de mêlée ou d'être à mains nues pour être exécutée. |
| **Movement** | **Mouvement** | Cette action vous permet de vous déplacer dans le cadre de son activation. |
| **Natural** | **Naturelle** | Les armes naturelles bénéficient de la formation aux armes naturelles. Les armes naturelles ne peuvent pas être lâchées. |
| **Non Combat** | **Hors combat** | Actions qui ne peuvent être effectuées qu'en dehors d'une rencontre de combat. |
| **One Hand** | **Une main** | Cette action nécessite de manier une arme à une main. |
| **Oversized** | **Surdimensionné** | Cet objet est surdimensionné ou autrement encombrant à manier, vous empêchant d'agir rapidement. Vous avez +2 malédictions à l'initiative tant que cet objet est équipé. |
| **Parrying** | **Parade** | Cette arme fournit un bonus à votre défense de parade. |
| **Projectile** | **Projectile** | Cette action nécessite l'utilisation d'une arme à projectile. |
| **Rallying** | **Ralliement** | Cette action offre une restauration en testant le seuil de folie d'un allié. |
| **Ranged** | **À distance** | Cette action nécessite de manier une arme à distance pour être exécutée. |
| **Reaction** | **Réaction** | Cette action ne peut être effectuée que pendant le tour d'une autre créature lorsque sa condition d'activation est remplie. |
| **Reload** | **Rechargement** | Cette arme nécessite un rechargement pour frapper. |
| **Reliable** | **Fiable** | Cette arme est digne de confiance et ne produit presque jamais de résultats spectaculaires mauvais. Le seuil d'échec critique de l'arme est réduit de 2. |
| **Returning** | **Retour** | Cette arme est conçue pour revenir à votre main après avoir été lancée. |
| **Shield** | **Bouclier** | Cette action nécessite de manier un bouclier pour être exécutée. |
| **Skill** | **Compétence** | Cette action implique un test de compétence ciblé ou opposé. |
| **Spell** | **Sort** | Cette action implique la sorcellerie. |
| **Summon** | **Invocation** | Cette action invoque, appelle ou crée d'une autre manière un combattant. |
| **Thrown** | **Lancée** | Cette arme est conçue pour être adaptée au lancer. |
| **Two Handed** | **Deux mains** | Cette action nécessite de manier une arme à deux mains pour être exécutée. |
| **Unarmed** | **À mains nues** | Cette action nécessite que vous soyez complètement à mains nues pour être exécutée. |
| **Unarmored** | **Sans armure** | Cette action nécessite que vous soyez sans armure pour être exécutée. |
| **Versatile** | **Polyvalent** | Cette arme peut être maniée à une ou deux mains. |
| **Weakened** | **Affaibli** | Cette action est plus faible que les attaques normales et a un bonus de dégâts de -2. |

### Catégories d'Équipement

| Terme original | Traduction | Description |
| :--- | :--- | :--- |
| **Accessory** | **Accessoire** | Objets d'équipement comme les vêtements ou les bijoux. |
| **Armor** | **Armure** | Pièces d'armure qui offrent une défense. |
| **Consumable** | **Consommable** | Objets qui sont consommés ou ont un nombre d'utilisations limité. |
| **Loot** | **Butin** | Objets divers, comme des trésors ou des ingrédients. |
| **Weapon** | **Arme** | Armes de combat. |

### Termes Techniques (Système de Feuille)

| Terme original | Traduction | Description |
| :--- | :--- | :--- |
| **Archetype** | **Archétype** | Un modèle de progression pour les capacités et les compétences d'un personnage. |
| **Ancestry** | **Ascendance** | L'origine biologique et culturelle d'un personnage. |
| **Background** | **Historique** | Le passé d'un personnage, qui lui confère des connaissances et des talents. |
| **Damage Type** | **Type de dégâts** | La nature des dégâts infligés (acide, feu, etc.). |
| **Defenses** | **Défenses** | Les défenses d'un personnage (Armure, Esquive, Volonté, etc.). |
| **Enchantment** | **Enchantement** | Le niveau d'enchantement d'un objet. |
| **Engagement** | **Engagement** | Le nombre d'ennemis qu'un personnage peut engager simultanément. |
| **Focus Points** | **Points de Concentration** | L'énergie mentale nécessaire pour des actions spéciales. |
| **Heroism** | **Héroïsme** | Une ressource représentant des prouesses héroïques. |
| **Quality** | **Qualité** | La qualité de fabrication d'un objet (Mauvaise, Standard, etc.). |
| **Skills** | **Compétences** | Les compétences que les personnages peuvent développer. |
| **Stride** | **Foulée** | La vitesse de déplacement par action. |
| **Talent** | **Talent** | Une capacité spéciale ou un don qu'un personnage peut acquérir. |
| **Taxonomy** | **Taxonomie** | La catégorie de créature à laquelle un adversaire appartient (Bête, Humanoïde, etc.). |
